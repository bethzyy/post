# 工具管理器使用指南

## 两种工具管理器

### 1. 命令行版工具管理器（新版）
**启动方式**:
- 双击 `启动工具管理器.bat`
- 或运行: `python tool_manager.py`

**特点**:
- 支持新的目录结构（5个分类）
- 交互式菜单
- 无需浏览器
- 适合快速访问

### 2. 网页版工具管理器
**启动方式**:
- 自动运行在 http://localhost:5000
- 直接在浏览器中打开该地址

**特点**:
- 可视化界面
- 一键运行工具
- 实时状态监控
- 适合可视化操作

## 使用方法

### 命令行版
1. 双击 `启动工具管理器.bat`
2. 选择分类（1-5）
3. 选择工具序号
4. 等待工具运行完成

### 网页版
1. 浏览器打开 http://localhost:5000
2. 点击"▶ 运行"按钮
3. 查看实时输出结果

## 快速启动

```bash
# 命令行版
python tool_manager.py

# 网页版（如果未运行）
python tool_manager.py web
```

## 注意事项

1. 确保Python 3.8+已安装
2. 确保API密钥已配置
3. 网页版需要Flask: `pip install flask`
4. 部分工具需要网络连接

## 故障排除

**问题**: 端口被占用
**解决**: 修改tool_manager.py中的端口号

**问题**: 中文乱码
**解决**: 使用批处理文件启动（已设置UTF-8编码）

**问题**: 工具运行失败
**解决**: 检查API密钥配置，查看错误信息
