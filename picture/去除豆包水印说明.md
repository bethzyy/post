# 去除"豆包AI生成"水印工具 - 使用说明

## 🎯 专门针对"豆包AI生成"水印

这个工具是**专门设计用来去除"豆包AI生成"水印**的优化版本。

### 水印特点
- **位置**: 通常在图片**右下角**
- **内容**: "豆包AI生成"这几个字
- **颜色**: 通常是白色或浅色半透明文字

---

## 🚀 使用方法

### 方式1: 通过工具管理器（推荐）

1. 访问: **http://localhost:5000**
2. 找到"节日图像生成"分类
3. 找到: **"去除'豆包AI生成'水印"**
4. 点击"启动"按钮

### 操作步骤

1. **选择图片** - 选择带"豆包AI生成"水印的图片
2. **设置输出目录** - 选择保存位置
3. **点击"开始去除水印"**
4. **等待处理** - 约5-10秒
5. **查看结果** - 只生成一张完美图片

---

## 🔍 工作原理

### 自动检测水印位置

工具会自动检测图片右下角的"豆包AI生成"文字：

1. **MSER文字检测** - 检测文字区域
2. **边缘检测** - 检测浅色文字
3. **右下角定位** - 重点检查右下角区域
4. **固定区域备选** - 如果检测失败，使用右下角固定区域

### 精确去除

```
原图 → 检测右下角水印 → 创建精确mask → Inpainting修复 → 完美去除
```

**关键**:
- ✅ 只修改水印区域
- ✅ 原图其他内容完全保留
- ✅ 智能填充水印位置
