# AI智能去水印工具 - 使用说明

## 工具简介

这是一个使用AI大模型重新生成图片来去除水印的智能工具。与传统去水印方法不同，本工具通过AI重新生成图片内容，可以完美去除各种类型的水印。

## 核心优势

### ✅ 完美去水印
- AI重新生成图片，水印完全消失
- 不残留任何水印痕迹
- 图片质量不受影响

### ✅ 支持复杂水印
- 文字水印
- Logo水印
- 半透明水印
- 复杂图案水印
- 任何类型的水印都能去除

### ✅ 多种AI模型
1. **Pollinations (推荐，免费)**
   - 完全免费使用
   - 生成速度快
   - 支持Flux模型
   - 无需API密钥

2. **DALL-E 3 (高质量)**
   - OpenAI官方模型
   - 生成质量最高
   - 细节最丰富
   - 需要API密钥

3. **Volcano/Seedream**
   - 字节跳动模型
   - 2K高分辨率
   - 中文优化
   - 需要API密钥

4. **Gemini 2K**
   - Google Gemini模型
   - 2K高分辨率
   - 质量优秀
   - 需要API密钥

## 操作步骤

### 1. 选择图片
- 点击"选择要去水印的图片"按钮
- 可以选择多张图片批量处理

### 2. 设置输出目录
- 点击"浏览"按钮选择输出目录
- 默认为当前工作目录

### 3. 选择AI模型
- **Pollinations**: 推荐使用，完全免费
- **DALL-E 3**: 需要配置OpenAI API密钥
- **Volcano**: 需要配置字节跳动API密钥
- **Gemini**: 需要配置Google API密钥

### 4. 输入图片描述（重要！）
在"图片内容描述"文本框中详细描述图片内容，例如：

**美食图片示例**:
```
一张精美的美食摄影照片，展示一道传统的中国菜肴，
色泽诱人，摆盘精致，高分辨率，专业摄影，
柔和的灯光，细节丰富，美食摄影风格
```

**风景图片示例**:
```
一张美丽的风景照片，青山绿水，蓝天白云，
自然风光，高分辨率，专业摄影，
色彩鲜艳，细节丰富
```

**人物照片示例**:
```
一张人物肖像照片，自然光线，表情生动，
背景虚化，高分辨率，专业摄影风格
```

**提示**:
- 描述越详细，AI生成的图片越接近原图
- 包含: 主体、风格、光线、分辨率等要素
- 可以参考原图的构图和内容

### 5. 开始处理
- 点击"开始AI去水印"按钮
- 等待AI生成新图片（可能需要10-60秒）
- 查看处理日志了解进度

### 6. 查看结果
- 处理完成后的图片保存在输出目录
- 文件名格式: `原文件名_ai_removed.扩展名`

## 注意事项

1. **API配置**
   - Pollinations无需配置，直接使用
   - DALL-E 3需要在`config.py`配置OpenAI API
   - Volcano需要在`config.py`配置字节跳动API
   - Gemini需要在`config.py`配置Google API

2. **处理时间**
   - Pollinations: 约10-30秒/张
   - DALL-E 3: 约15-40秒/张
   - Volcano: 约20-50秒/张
   - Gemini: 约15-45秒/张

3. **图片描述**
   - 描述越详细，效果越好
   - 包含图片的主题、风格、光线等要素
   - 可以添加"无水印"、"干净的画面"等关键词

4. **批量处理**
   - 可以一次选择多张图片
   - 建议每批不超过10张
   - 处理时间会累加

## 常见问题

**Q: AI生成的图片和原图不一样？**
A: AI是重新生成图片，不会100%和原图相同。描述越详细，越接近原图。

**Q: 哪个模型效果最好？**
A: DALL-E 3质量最高，Pollinations免费且速度快。

**Q: 如何提高生成质量？**
A: 详细描述图片内容，包括主体、风格、光线、背景等要素。

**Q: 可以处理哪些格式的图片？**
A: 支持JPG、PNG等常见格式。

**Q: 生成失败怎么办？**
A: 检查API配置，或尝试使用Pollinations（无需API）。

## 技术说明

### 工作原理
1. 用户上传带水印的图片
2. 用户描述图片内容
3. AI根据描述重新生成图片
4. 新生成的图片无水印，内容相似

### 支持的AI模型
- **Pollinations**: Flux模型，免费使用
- **DALL-E 3**: OpenAI，需API密钥
- **Seedream**: 字节跳动，2K分辨率
- **Gemini**: Google，2K分辨率
