# 今日头条文章生成器 - 使用说明

## ✨ 新功能

**AI驱动的智能文章生成**,支持用户输入任意主题!

### 🎯 主要改进

1. **用户自定义主题** - 不再局限于预设主题,可输入任何主题
2. **AI智能生成** - 使用智谱GLM-4模型生成高质量文章
3. **可选择字数** - 支持1500/2000/2500字三种长度
4. **精美HTML输出** - 自动生成排版美观的HTML页面
5. **Markdown格式** - 同时保存MD格式,方便编辑

### 📝 使用方法

#### 运行工具

```bash
cd article
python toutiao_article_generator.py
```

#### 交互式输入

**步骤1: 输入主题**
```
请输入主题 (输入 'q' 退出): 过年回老家
```

**示例主题**:
- 过年回老家
- 职场新人必看
- 健康养生小贴士
- 理财投资心得
- 教育孩子感悟
- 情感关系建议

**步骤2: 选择字数**
```
请选择文章长度:
  1. 1500字左右 (快速阅读)
  2. 2000字左右 (标准长度)
  3. 2500字左右 (深度文章)

请选择 (默认为2):
```

**步骤3: 等待生成**
- 工具会调用AI生成文章
- 显示生成进度
- 自动保存并打开预览

### 📂 输出文件

生成后会创建两个文件:

1. **Markdown文件** - `今日头条文章_{主题}_{时间戳}.md`
   - 纯文本格式
   - 方便后续编辑
   - 可导入其他平台

2. **HTML文件** - `今日头条文章_{主题}_{时间戳}.html`
   - 精美排版
   - 自动在浏览器中打开
   - 可直接复制到今日头条

### 🎨 文章特点

AI生成的文章包含:

✅ **吸引人的标题**
- 使用数字+疑问/对比
- 字数15-25字
- 高点击率结构

✅ **引人入胜的开头**
- 制造悬念或共鸣
- 引发读者兴趣

✅ **3-5个核心要点**
- 真实案例
- 数据支撑
- 实用建议

✅ **感人的结尾**
- 情感共鸣
- 行动号召
- 互动引导

### ⚙️ 技术细节

**AI模型**: 智谱GLM-4-Flash
- 快速响应
- 高质量输出
- 成本较低

**参数设置**:
- Temperature: 0.8 (创造性)
- Max Tokens: 4000
- Top P: 0.9

### 🔧 故障排除

**问题1: AI调用失败**
```
[ERROR] 无法初始化AI客户端
```
解决: 检查`config.py`中的`ZHIPU_API_KEY`配置

**问题2: 生成内容太短**
解决: 选择更大的字数选项(2500字)

**问题3: HTML无法打开**
解决: 手动双击HTML文件打开
